# Molecule WorkFlow Pro - Backend

FastAPI backend with MongoDB integration.

## Quick Start

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run server:
```bash
python main.py
```

3. Access API docs:
- Swagger: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc
